How to add media:

📷 PHOTOS — use Imgur:
1. Go to imgur.com in Safari
2. Tap New Post → upload your photo
3. Once uploaded tap the image → copy the direct .jpg link
4. Paste above & select Photo

🎬 VIDEOS — use YouTube Unlisted:
1. Open YouTube app → tap + → upload video
2. Set visibility to Unlisted
3. Copy the youtu.be link
4. Paste above & select Video

How to share from Google Drive:
Open Google Drive → tap file → Share → Change to "Anyone with the link" → Copy link → paste above
